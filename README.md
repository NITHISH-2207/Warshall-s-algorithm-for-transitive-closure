# 🚀 Warshall’s Algorithm – Transitive Closure

## 📖 Overview
This project presents an implementation of **Warshall’s Algorithm** to compute the **transitive closure** of a directed graph. The algorithm determines the reachability between all pairs of vertices by identifying both direct and indirect paths using a dynamic programming approach.

A simple web-based interface is developed to allow users to input an adjacency matrix and visualize the step-by-step transformation of the matrix until the final transitive closure is obtained.

---

## 🎯 Objectives
- To implement Warshall’s Algorithm for graph reachability
- To understand transitive closure using adjacency matrices
- To visualize step-by-step matrix updates
- To analyze the time complexity of the algorithm
